Introduction

What is required?
  • Build a social networking web application able to gather user profiles from different platforms ( like LinkedIn, Last.fm, Instagram, WordPress, etc.) to build and manage the social network for a certain user.
  • The user can also interact through a web extension which can also offer friend suggestions based on other relevant data ( for example person P is friend with you on Facebook but he/she also has a GitHub account, and you've also got an account. Would you like to add P in your GitHub circle?).
  • There will also be available advanced search based on certain criteria (name, age, location, hobbies, etc.).
  • Social graphs are available for download in FOAF format.
  • There will also be available a preview of the graph in raster (PNG) and vectorial (SVG) format.
  • Bonus: 3D Preview using WebGL.

Project structure

On a first quick view we imagined a flow for the user and decided that we need the following pages:

  • A register page;
  • Login page;
  • Social profiles pages (where the logged user will enter his profiles from other social networking platforms);
  • Graph page (where user can preview and download his social networking graph, and search other users).

  • Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. With Cascading Style Sheets (CSS) and JavaScript, it forms a triad of cornerstone technologies for the World Wide Web

CSS

  • CSS , by Håkon Wium-Lie ; first proposed in 1994 (accessed on 25 Oct 2019 ) .

  • Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in a markup language like HTML. CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.

JavaScript

  • JS , by National Center for Supercomputing Applications, University of Illinois at Urbana-Champaign — Urbana and Champaign, Illinois, U.S. ; published in 1993 (accessed on 25 Oct 2019 ) .

  • JavaScript, often abbreviated as JS, is a high-level, interpreted programming language. It is a language which is also characterized as dynamic, weakly typed, prototype-based and multi-paradigm. Alongside HTML and CSS, JavaScript is one of the three core technologies of the World Wide Web.
